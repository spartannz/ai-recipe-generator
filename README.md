AI Recipe Generator

A mobile application that uses the OpenAI API to generate recipe ideas based on ingredients the user enters.
Built with React Native (Expo) on the frontend and Node.js + Express on the backend.

⸻

Features
	•	Generate complete recipes using AI
	•	Includes title, summary, ingredients list, and cooking steps
	•	Generate 3 similar recipes based on the original
	•	Simple, clean user interface

⸻

Tech Stack
	•	React Native (Expo)
	•	Node.js + Express
	•	OpenAI API
	•	Git & GitHub
